Fractal Forge
=============
Инструменты для создания и жесткой оптимизации процедурной растительности в Blender 5.2+
Вкладка в N-панели: FF


ТРЕБОВАНИЯ
----------
- Blender 5.2.0 или выше
- Никаких душных зависимостей - просто установи и используй!


УСТАНОВКА
---------
1. Скачай .zip архив fractal_forge
2. В Blender: Edit > Preferences > Add-ons > Install
3. Выбери файл и включи Fractal Forge
4. Вкладка FF появится в N-панели (нажми N во вьюпорте)


ЯЗЫК ИНТЕРФЕЙСА
---------------
Edit > Preferences > Add-ons > Fractal Forge > Preferences
Выбери English или Русский


ИНСТРУМЕНТЫ
-----------

КЛАСТЕРЫ
  Чистая геометрия без альфа-масок (прощай Overdraw).
  Хранят системные атрибуты (fractal_id и ColorMask) для LODS, Мясника и движка.

  Трава     - Патч травы с дуговым изгибом, UV и ColorMask
  Клевер    - Патч клевера с UV и ColorMask
  Ромашка   - Патч ромашки с UV и ColorMask
  Щебенка   - Патч камней с UV
  Облака    - Процедурные облака с UV и ColorMask

ЮНИТ
  Блобы - Натягивает процедурный генератор Ghibli-кустов на любой меш для дальних планов.

ИНСТРУМЕНТЫ > LODS
  Два режима спасения FPS:
  Кластеры: удаляет геометрию целиком по ID (сохраняет 2D-силуэт).
  Юниты: классическая жесткая децимация через копии.

ИНСТРУМЕНТЫ > АВТО-НОРМАЛИ ЛИСТВЫ
  Оборачивает листву в fog proxy и крадет сферические нормали для аниме-стиля ^_____^
  (Важно: выдели меш перед запуском).

ИНСТРУМЕНТЫ > МЯСНИК
  Рубит кластер на независимые травинки по fractal_id и ставит Origin строго в корень. 
  Без этого WPO ветер в Unreal Engine оторвет траву от земли! (⌐■_■)

TOON SHADING (CEL-SHADING)
  Ультимативный аниме-шейдер. Вшивает математику освещения напрямую в ноды материала, 
  считает каскады и тени (Raycast) для каждой лампочки отдельно. Свет ложится ровными полосами.

ТЕХНИЧЕСКИЕ ДЕТАЛИ
------------------
- Генерация на базе Geometry Nodes.
- Связь нод и Python через атрибуты (fractal_id, ColorMask).
- Динамическое создание нодовых групп через API.
