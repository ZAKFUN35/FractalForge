Fractal Forge
=============
Инструменты для создания и оптимизации процедурной растительности в Blender 4.0+
Вкладка в N-панели: FF


ТРЕБОВАНИЯ
----------
- Blender 4.0.0 или выше
- Никаких дополнительных зависимостей -- просто установи и используй!


УСТАНОВКА
---------
1. Скачай .zip архив fractal_forge
2. В Blender: Edit > Preferences > Add-ons > Install
3. Выбери файл и включи Fractal Forge
4. Вкладка FF появится в N-панели (нажми N во вьюпорте)


ЯЗЫК ИНТЕРФЕЙСА
---------------
Edit > Preferences > Add-ons > Fractal Forge > Preferences
Выбери English или Русский


ИНСТРУМЕНТЫ
-----------

КЛАСТЕРЫ
  Объекты процедурной растительности создаются полностью с помощью геометрических нод. 
  Каждый объект хранит системные атрибуты, необходимые для работы инструментов LODS и Мясник.

  Трава     - Патч травы с дуговым изгибом лезвий, UV и атрибутом ColorMask
  Клевер    - Патч клевера с UV и атрибутом ColorMask
  Ромашка   - Патч ромашки с UV и атрибутом ColorMask
  Щебёнка   - Патч с камнями с UV

ЮНИТ
  Блобы  - Преобразует выделенный объект в блобы полностью с помощью геометрических нод.

ИНСТРУМЕНТЫ > LODS
  Генерирует уровни детализации (LOD) только для объектов, созданных инструментами из категории КЛАСТЕРЫ. 
  Создаёт иерархию коллекций с обрезанными версиями меша.

ИНСТРУМЕНТЫ > АВТО-НОРМАЛИ ЛИСТВЫ
  Переносит нормали с туманного прокси-меша на геометрию листвы для того, чтобы сделать сферические нормали для стилизации.

  Важно: выдели меш листвы перед запуском.

ИНСТРУМЕНТЫ > МЯСНИК
  Разделяет сгенерированный кластер на отдельные самостоятельные меши (например, делит патч травы на все травинки как отдельные объекты).


СТРУКТУРА ФАЙЛОВ
----------------
fractal_forge/
├── nodes/
│   ├── blob.py           <- Нода-группа Blob
│   ├── chamomile.py      <- Нода-группа Chamomile
│   ├── clover.py         <- Нода-группа Clover
│   ├── grass.py          <- Нода-группа Grass
│   ├── macadam.py        <- Нода-группа Macadam (камни)
│   └── __init__.py
├── operators/
│   ├── autonormals.py    <- Оператор Авто-нормали листвы
│   ├── butcher.py        <- Оператор Мясник
│   ├── clusters.py       <- Операторы создания кластеров
│   ├── lod.py            <- Оператор создания LOD
│   ├── unit.py           <- Оператор конвертации в блобы
│   └── __init__.py
├── __init__.py           <- Точка входа аддона и bl_info
├── i18n.py               <- Локализация (EN / RU)
├── panels.py             <- Разметка UI в N-панели
├── properties.py         <- Настройки аддона и группы свойств
├── README.txt            <- Английская версия
└── ЧИТАЙМЕНЯ.txt         <- Этот файл


ТЕХНИЧЕСКИЕ ДЕТАЛИ
------------------
- Полностью процедурная генерация на базе Geometry Nodes
- Связь между нодами и Python-операторами через кастомные атрибуты (fractal_id, ColorMask)
- Динамическое создание, подключение и очистка нодовых групп через Blender Python API


ПРОТЕСТИРОВАНО НА
-----------------
- Blender 4.5.4 LTS
- Windows
